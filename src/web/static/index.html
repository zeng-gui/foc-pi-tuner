4. 调参指南

常见问题

  • • 振荡：增益过高 → 减小Kp
  • • 超调大：阻尼不足 → 增大Kp或减小Ki
  • • 响应慢：带宽不足 → 增大Kp
  • • 稳态误差：Ki不足 → 增大Ki

调试步骤

  1. 先调电流环，确保稳定
  2. 再调速度环，从小增益开始
  3. 观察阶跃响应波形
  4. 根据表现微调参数
